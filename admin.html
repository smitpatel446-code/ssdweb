📖 Cloudinary Setup Guide

1
Create a free Cloudinary account
Go to cloudinary.com → Sign up free. Your Cloud name is shown on your dashboard (e.g. silverstar).
2
Create an Upload Preset
In Cloudinary → Settings → Upload → Scroll to "Upload presets" → Click "Add upload preset".
Set Signing mode = Unsigned. Give it a name like silverstargallery.
Under "Tags" add the tag silverstargallery so every upload gets tagged automatically. Save.
3
Enable tag-based listing
In Cloudinary → Settings → Security → tick "Resource list" to allow public tag listing. Save.
4
Update this admin file
Open admin.html in a text editor. Find the three lines near the top of the script:
var CLOUD_NAME = 'YOUR_CLOUD_NAME';
var UPLOAD_PRESET = 'YOUR_UPLOAD_PRESET';
var ADMIN_PASSWORD = 'YOUR_PASSWORD';
Replace the placeholder values with your own.
5
Update index.html
Open index.html. Find the gallery script section and update:
var CLOUD_NAME = 'YOUR_CLOUD_NAME'; — same cloud name.
var TAG = 'silverstargallery'; — must match the tag in your preset.
6
Deploy to Netlify
Upload both index.html and admin.html to Netlify.
Your main site is at yourdomain.com.
The admin is at yourdomain.com/admin (Netlify serves admin.html at /admin — make sure you rename the file to match the path or configure a redirect if needed).
Tip: Create a folder named admin containing index.html (rename admin.html to index.html inside that folder) so the URL becomes yourdomain.com/admin/.
7
Upload & enjoy!
Open /admin, enter your password, upload photos and video. They will appear on the main site within seconds — no rebuild needed.
